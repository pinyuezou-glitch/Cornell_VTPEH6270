## Project Description
This repository contains coursework for VTPEH 6270, focusing on data exploration and statistical simulation using R. The project demonstrates basic data analysis workflows and reproducible research practices using GitHub.

---

## Research Objectives
The objectives of this project are:
- To explore and summarize a dataset using descriptive statistics and visualization  
- To simulate data using statistical distributions in R  
- To demonstrate reproducible workflows using Git and GitHub  

---

## Data Source and Description
The data used in this project includes publicly available or course-provided datasets. For demonstration purposes, built-in R datasets (such as `mtcars`) and/or provided datasets are used.

---

## Deliverables
The repository includes:
- Data exploration script: `scripts/data_exploration.Rmd`  
- Data simulation script: `scripts/data_simulation.Rmd`  
- Rendered reports located in the `reports/` folder
